<p>You can execute following code on terminal to change a folder to an iso.  </p>
<blockquote>
<p>hdiutil makehybrid -o ~/Desktop/image.iso ~/path/to/folder/to/be/converted -iso -joliet</p>
</blockquote>
<p>That&#8217;s it. </p>
