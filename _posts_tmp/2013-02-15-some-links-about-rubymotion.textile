<p>Ruby motion is a toolchain for iOS. You can develop native iOS application with ruby language via RubyMotion. RubyMotion has rake based workflow. And it is easy to use. Use link <a href="http://www.rubymotion.com/" target="_blank">[1]</a> to start. There ara a documentation for developers and <a href="http://www.rubymotion.com/developer-center/guides/getting-started/" target="_blank">a hello world application</a>. You can follow the news with <a href="http://blog.rubymotion.com/" target="_blank">blog</a>. You can read some articles, reviews, like these <a href="http://merbist.com/2012/05/04/macruby-on-ios-rubymotion-review/" target="_blank">[2]</a>  <a href="http://37signals.com/svn/posts/3432-why-i-loved-building-basecamp-for-iphone-in-rubymotion" target="_blank">[3]</a>, <a href="http://arstechnica.com/business/2012/05/exclusive-building-ruby-ios-applications-with-rubymotion/" target="_blank">[4]</a>, <a href="http://aelogica.com/development/rubymotion-review/" target="_blank">[6]</a>. After that you can follow the some repos from <a href="https://github.com/rubymotion" target="_blank">rubymotion github community</a>.  You find some samples following <a href="https://github.com/HipByte/RubyMotionSamples" target="_blank">[5]</a> link.</p>
